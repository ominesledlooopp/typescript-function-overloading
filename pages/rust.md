# TypeScript & Rust Syntax

___

## TypeScript
###### function declaration

___

```typescript

function typescript_( name: string ): number {

  if( name === 'Jack') {
     return 1
  }

  return 0
}

```

## Rust
###### function declaration

___

```rust

fn rust_( name: &str ) -> i8 {

  if name == "Jack" {
      return 1
  }

  return 0
}

```
